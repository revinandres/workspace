Base de datos

¿Qué es una base de datos?

Una base de datos es un conjunto organizado de información o datos que se almacenan para consultarlos y gestionarlos fácilmente.

¿Para qué sirve una BD?

Una base de datos (BD) es un sistema donde se guardan datos relacionados entre sí, permitiendo acceder a ellos y utilizarlos eficientemente.

¿Qué es un CRUD? y definición de cada letra

Un CRUD es un conjunto de cuatro operaciones básicas que se realizan en una base de datos.

C – Create (Crear): agregar nuevos datos.
R – Read (Leer): consultar los datos.
U – Update (Actualizar): modificar datos existentes.
D – Delete (Eliminar): borrar datos de la base de datos.

¿Qué es un campo?

En una base de datos, un campo es la unidad más pequeña de información dentro de un registro. Por ejemplo, Nombre, Edad o Correo electrónico.

¿Qué es un registro?

Un registro es un conjunto de campos relacionados que representan información completa sobre un elemento. Ejemplo: Nombre, Edad y Dirección de una persona.

¿Qué es un dato?

Un dato es la unidad básica de información, como un número, texto o fecha. Ejemplo: “Juan” o “25”.

Diagrama Entidad-Relación

Un diagrama Entidad-Relación muestra las entidades, sus atributos y las relaciones entre ellas en una base de datos.
